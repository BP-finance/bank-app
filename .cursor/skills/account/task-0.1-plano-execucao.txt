1.1 Nome e local

1.2 Estrutura mínima (enxuta, alinhada ao padrão do projeto)

    src/features/account-center/
      index.ts                          <- API pública (export da screen e tipos se houver)
      presentation/
        screens/
          AccountCenterScreen.tsx       <- Screen principal da superfície (estrutural, não tela completa)

    Justificativa da estrutura:
    - Outras features (security, auth) expõem screens via index e têm presentation/screens.
    - Nesta task NÃO criar: store, hooks, services, types (exceto se um type mínimo for necessário para o index).
    - A screen será placeholder mínimo: mesma experiência atual de app/user.tsx (título "Perfil do Usuário" + LogoutButton), mas vivendo na feature, para app/user.tsx ficar só delegando.

1.3 Conteúdo mínimo de AccountCenterScreen.tsx
    - Componente que renderiza a mesma UI atual da rota /user: título "Perfil do Usuário" + LogoutButton.
    - Import de LogoutButton de @/src/components/debug/LogoutButton (não reimplementar logout; AUTH permanece dono).
    - Sem lógica de domínio nova; sem AUTH/SECURITY/PIX internos; sem payloads inventados.
    - Objetivo: materializar que a superfície real pertence à feature; a rota só delega.

1.4 Conteúdo mínimo de index.ts
    - export { AccountCenterScreen } from "./presentation/screens/AccountCenterScreen";
    - Nada mais por enquanto (sem export de store/hooks de SECURITY/AUTH aqui; consumo virá em tasks futuras).

3.1 Opção escolhida: redirecionamento fino para /user

3.2 Arquivo: app/settings/settings.tsx

    Antes: componente SettingsScreen com View, Text "Configuraçoes", styles.
    Depois:
    - Usar useRouter() + useEffect (ou Redirect do expo-router se existir e for adequado) para redirecionar para "/user".
    - Se expo-router oferecer <Redirect href="/user" />, usar isso no render (mais declarativo); senão, useEffect(() => { router.replace("/user"); }, []) com return null ou um mínimo loading.
    - Comentário no topo: deixa explícito que esta rota não é área concorrente; a superfície oficial é /user (account-center).

    Verificação no código: checar se expo-router tem Redirect ou se o padrão do app é router.replace em useEffect.
